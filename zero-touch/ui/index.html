Zero Touch Provisioning

Self-hosted automation for mixed fleet devices.

API Endpoints

  • POST /enrollment/tokens — create enrollment token (requires X-API-Key)
  • POST /enrollment/register — agent registers using token
  • POST /blueprints — create blueprint
  • POST /workflows/devices/{id} — apply blueprint to device

Agent Bootstrap (Linux/macOS)

curl -s https://api.localhost/enroll?token=YOUR_TOKEN -o bootstrap.sh
chmod +x bootstrap.sh
./bootstrap.sh

Agent Bootstrap (Windows)

iwr -useb https://api.localhost/enroll?token=YOUR_TOKEN -OutFile bootstrap.ps1
powershell.exe -ExecutionPolicy Bypass -File .\\bootstrap.ps1
